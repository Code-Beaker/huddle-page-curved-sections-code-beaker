## Process

The full command line process is given below

### Create Vite App

```sh
npm create vite@latest -- --template vanilla app
```

```sh
cd app
```

#### Install dependencies and packages

```sh
npm install

npm add sass -D

npm install validator
```

### Import the files with JavaScript

![alt text](image-2.png)

### Preview the website

